Volet 支持中国大陆用户使用身份证进行注册，部分用户可能需要提供英文地址才能完成KYC认证。Volet还提供实体卡，支持在ATM机取现，但使用VPN可能会导致登录问题。

## Volet 出入金方式

Volet 支持内部用户间的免费转账，并可以将加密货币转换为欧元或美元，随后提现到银行账户。支持的USDT网络包括ERC20、TRC20、BEP20等。
